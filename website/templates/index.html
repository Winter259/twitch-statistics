The Ocellus Project

Public twitch stream statistics for Elite: Dangerous and Planet Coaster for all to see and use

The Ocellus project is a data collection, processing and visualisation project aimed at making twitch.tv stream data for a specific set of games available for users to see. All data gathered is available directly from twitch's public API which is then stored and processed through a number of scripts to produce some sort of understandable output.

Aim of the project

As a viewer

  • feedback Discover new or popular streamers for your game of choice.
  • description Find out which stremaers have the best or most consistent exposure to better market for giveaways or sponsorships.

As a streamer

  • trending_up Track your viewer and follower count over time, see how it varies according to what time you stream.
  • assessment Compare yourself to other streamers with similar interests, presenting the opportunity for collaboration.
